# Logging with CloudTrail

AWS CloudTrail主要关注“谁在何时对您的AWS账户做了什么”，侧重于审计和合规性；而AWS CloudWatch则聚焦于“您的AWS资源和应用程序的运行状况如何”，侧重于性能监控和运营健康。存储在S3存储桶中的详细、不可变的审计日志。
